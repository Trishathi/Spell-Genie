# 🧙 SpellGenie

SpellGenie is a simple Python GUI application that checks and corrects spelling mistakes using the **TextBlob** library. Users can enter a word or text, click the **Check** button, and instantly see the corrected spelling.

## ✨ Features

- User-friendly graphical interface built with Tkinter
- Automatic spelling correction using TextBlob
- Instant display of corrected text
- Lightweight and easy to use
- Beginner-friendly Python project

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI Framework)
- TextBlob (Natural Language Processing Library)

## 📋 Prerequisites

Make sure you have Python installed on your system.

Check your Python version:

```bash
python --version

📦 Installation

Clone the repository:
git clone https://github.com/your-username/spellgenie.git
Navigate to the project folder:
cd spellgenie
Install the required dependency:
pip install textblob
(Optional) Download TextBlob corpora:
python -m textblob.download_corpora

▶️ Running the Application

Run the Python file:

python spellgenie.py

The SpellGenie window will open, allowing you to enter text and check its spelling.

📸 Application Workflow
Enter a word or sentence in the input field.
Click the Check button.
The application processes the text using TextBlob.
The corrected text is displayed on the screen.
